## ❤️ CardioPredict - Heart Disease Prediction System

A Machine Learning-powered web application that predicts the likelihood of heart disease based on patient health parameters. The project includes data analysis, model comparison, model training, and an interactive Streamlit application for real-time predictions.

### 🌐 Live Demo

🔗 **Deployed App:** https://heart-disease-analysis-hasini.streamlit.app/

---

### 📌 Project Overview

Heart disease is one of the leading causes of death worldwide. Early prediction can assist healthcare professionals in identifying high-risk patients.

This project uses Machine Learning algorithms to analyze patient medical information and predict whether a patient is likely to have heart disease.

The application also provides interactive visualizations to help users better understand the dataset and model performance.

### 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/heart-disease-prediction.git
```

Move into the project folder

```bash
cd heart-disease-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
